Step 2   Install external packages (LibPng, Zlib, OpenSSL, Xerces, STL)


2 a      LIBPNG and ZLIB

2 a 1   Download and extract LibPng
JDFTools uses PNG Library version 1.0.9 (lpng109.zip).    !!! Note
The latest version of LibPng can be found at the PNG home page: http://www.libpng.org/pub/png or at ftp://ftp.uu.net/graphics/png/src
Extract  Lpngxxx.zip  to  .\external\LipPng
If neccesary, rename the directory in .\external\libpng
2 a 2   Download and extract Zlib.zip
JDFTools uses ZLIB version 1.13 (zlib113.zip).     !!! Note
Zlib should be available at the same place that libpng is.
If not, it should be at ftp://ftp.uu.net/graphics/png/src or at ftp://ftp.uu.net/pub/archiving/zip/zlib
Extract  Zlib.zip  to  .\external\Zlib
If neccesary, rename the directory in .\external\zlib
2 a 3   Create DLLs
2.a.3.1 Open Workspace "external\libpng\projects\msvc\libpng.dsw"

2.a.3.2 Open "Menu -> Build -> Set Active Configuration..." and select "libpng Win32 DLL" configuration

2.a.3.3 Run "Menu -> Build -> Clean". Ignore error messages ... (Could not find the file ...)

2.a.3.4 Open "FileView -> zlib files -> Header Files" in workspace window and select "Properties" of zlib.h.
"Persist as" should be "..\..\..\zlib\zlib.h"

2.a.3.5 Select "Menu -> Build -> Rebuild all". Creates: zlib.lib, zlib.dll and libpng13.lib, libpng13.dll

2.a.3.6 Repeat Step 2.a.3 for "libpng Win32 DLL Debug" ,"libpng Win32 Lib" and "libpng Win32 Lib Debug" configurations (see 2.a.3.2)
