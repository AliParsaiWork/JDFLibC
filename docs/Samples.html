Sample Applications


JDFTestWrapper project you can find
using MSVC6 here:  ./Projects/Win32/Vc6/JDFTestWrapper.dsw
using MSVC7.1 here:./Projects/Win32/Vc7/JDFTestWrapper.sln
It contains the sample applications that are described below. The program JDFExample is the best starting point, as it is maintained and documented. The other samples compile and have been used for concept tests and providing the examples in the JDF specification. A set of JDF Wrapper Examples is provided.
In Menu "Sample Applications" there is also some auxiliary material, based on JDFExample

JDFExample

A set of simple examples how to create a JDF and/or extract information from it.

CheckJDF

A console application that uses the typesafe validator routines to check a JDF.

JDFSchema

A console application that generates the typesafe classes from the JDF schema using the classes from the WrapperCore package.

SpawnJDF / MergeJDF

Console applications that allow spawning and merging of JDF.

BookJDF

A console application that generates a Product Intent of a set of books in one JDF.

AddJDF

A console application that appends information to a JDF.

EatJDF

A console application that emulates a JDF device.

PrintTalk

A console application that emulates a PrintTalk rfq/quote/invoice cycle. (not maintained)
