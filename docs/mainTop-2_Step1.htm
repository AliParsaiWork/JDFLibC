Step 1   Unpack JDFOpenSrc

1 a     Unpack the downloaded JDF open source zip file to your prefered root directory.
1 b     Create the following directory structure:
  • JDFOpenSrc
    • Build
    • docs
      • jdf
      • jdfwrapper
      • jdfwrappercore
      • jdfwrapperexamples
    • doxygen
    • external
      • libpng
      • zlib
      • openssl
      • xerces
      • stlport
    • Projects
    • src
    • tests


Note: Stlport directory for Visual Studio C++ V7.1 may be omitted
