Installation procedure for the CIP4 JDF Open Source Library

Version Date: 09.01.2002
Updated: 04.05.2005

Requirements

  • Microsoft Visual Studio C++ 6.0 or Microsoft Visual Studio .NET C++ 7.1
  • Perl Interpreter e.g. http://downloads.activestate.com/ActivePerl/Windows/5.6/ActivePerl-5.6.1.630-MSWin32-x86.msi
  • 1-2 GB free space on disk
  • recommendation: fast processor, a lot of memory, if you want to build the project files in an acceptable lapse of time
