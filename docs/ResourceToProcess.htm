How to append a resource to the process (JDFNode)


To append a resource to the process, use AppendMatchingResource (const WString& yourResource, EnumProcessUsage processUsage) method and typesafe attribute/element setters.
To clarify what attributes and elements may have your resource, see JDF Specification (Release 1.2) chapter 7 “Resources”, their implementation you can find in JDFWrapper JDFResources in a corresponding package.
Generic attributes and elements are defined in chapter 3.6 “Resources”, implemented in JDFCore JDFResource package.

In the following example we have a process named productNode that is a Product Intent described as a JDFNode and append it with two resources (one input, one output).
See Example 2


1) We define the complete output Component (output resource):

JDFComponent compBrochure=productNode.AppendMatchingResource(JDFStrings::elm_Component,JDFNode::ProcessUsage_AnyOutput);


2) now we can add all kinds of attributes to compBrochure e.g.:

compBrochure.SetComponentType(JDFComponent::ComponentType_FinalProduct);
compBrochure.SetDescriptiveName("complete 16-page Brochure");
compBrochure.SetAmount(10000);


3) Add BindingIntent input resource:

JDFBindingIntent bInt=productNode.AppendMatchingResource(JDFStrings::elm_BindingIntent,JDFNode::ProcessUsage_AnyInput);
